Now let's fix up that boring list by adding a new class our <li> elements. To add the class 'pretty' (from our required sample.css file), open your broswer console window (In chrome, right click the window and select 'inspect'). Try this command:

$g('li').addClass('pretty')

Cool, we added a css class to all elements with an <li> tag. Dang, that looks pretty! Almost too pretty... let's make thinks more serious by running:

$g('li').parent().append('<p>hey settle down!</p>')

Awesome! we've create a new <p> tag DOM Node element and appended it to our <ul> Nice. But on second thought, let's keep the party going and remove that crumudgeonly <p> tag by running:

$g('p').remove()

Nice, now we can party. Try out some of the other DOM manipulation functionality, go wild! For more information, check out the documentation at https://github.com/SamStuckey/jGreat
